# SPC Backend Workbook Folder

Place the default SPC workbook here with this filename:

```text
SPC_Backend.xlsx
```

When the operator clicks **DefectSPC** or **SPC Excel**, the application opens that workbook directly instead of opening a folder or asking every time.

If the workbook has a different name or location, use **Link SPC Excel** / **Link File** once in the GUI to save the exact file path.
